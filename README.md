# 📚 Estudos Java: Fundamentos e Ecossistema
Este repositório foi criado para documentar minha jornada de aprendizado na linguagem Java, partindo dos conceitos fundamentais até a construção de APIs robustas com o ecossistema Spring.

# 🚀 Objetivo
O foco principal é consolidar o conhecimento em Programação Orientada a Objetos (POO), manipulação de dados e integração com ferramentas de mercado para desenvolvimento backend.

## 🛠️ Pré-requisitos e Instalação

Para rodar este projeto localmente, você precisará do **JDK 21**. Siga os passos abaixo:

1. **Download:** Acesse o [site da Oracle](https://www.oracle.com/java/technologies/downloads/#jdk21-windows).
2. **Sistema:** Escolha a versão correspondente ao seu Sistema Operacional (ex: Windows x64 Installer).
3. **Instalação:** Execute o instalador, avance as etapas (Next) e confirme se a instalação foi concluída na pasta `C:\Program Files\Java`.
4. **Verificação:** Abra o terminal e digite `java -version`.
